Class name: Scheduler
Parent Class: None
Subclass: BackToBackScheduler, ProfPreferenceScheduler, DurationScheduler, SleepScheduleScheduler

Responsibility:
- Store instances of class Course
- Has an abstract method to compute possible course session combinations, to be implemented by child classes
- Create and store instances of class Timetable, where each Timetable presents a possible course combination based on the user's given preference
- Print every Timetable stored

Collaborator:
* Course
* Timetable
